1. Frozen Lake.
- These are a set of Jupyter Notebooks that can be run online through an Anaconda Installation of Jupyter
- Since it uses OpenAI gym:
        - Install Gym 
        - Make sure you update the following line to match the location of your installed Gym environment.
		sys.path.append('/home/gabe/.local/lib/python3.8/site-packages')

2. Occupancy.
- These are a set of python files that can be executed via the main.py class.  
- Use Python 3.8 environment
- Visual Studio Code recommended.
